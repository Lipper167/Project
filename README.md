# Project
Nutrition diet checker
This project helps in checking the nutritional values of different foods our software classifies food into junk food or healthy using health score. Though it may give some unwanted outcomes due to api providing incomplete information sometimes but it works great with common foods which help in maintaining good health.

To run the code you need to have the USDA API which you can get from https://fdc.nal.usda.gov/api-key-signup
